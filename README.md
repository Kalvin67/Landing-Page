# VORTEX X Pro — Product Landing Page

A responsive single-page product landing page for a gaming mouse built as part of a web development activity.

## 🔴 Live Site
[View on GitHub Pages](https://kalvin67.github.io/Landing-Page/)

## 📋 About
The VORTEX X Pro landing page is a fully responsive, single-page website showcasing a fictional high-performance gaming mouse. It was built using only HTML5, CSS3, and Vanilla JavaScript — no frameworks.

## 📌 Sections
- **Navigation Bar** — Fixed responsive nav with smooth scrolling and mobile hamburger menu
- **Hero Section** — Catchy headline, animated SVG product illustration, and floating stat badges
- **Stats Bar** — Key specs displayed in a clean grid (DPI, Polling Rate, Weight, Battery)
- **Features Grid** — 6-card grid showcasing the product's unique benefits
- **Interactive Section** — RGB color picker and live countdown timer
- **Specs & Pricing** — Full technical specs table with pricing tiers
- **Footer** — Social media links, navigation links, and copyright info

## ✅ Requirements Met
- HTML5 semantic tags (`<header>`, `<section>`, `<footer>`, `<nav>`)
- CSS3 Flexbox and Grid for layout
- Custom Google Fonts (Bebas Neue, Nunito, JetBrains Mono)
- Responsive design with media queries
- JavaScript interactive elements:
  - Toggleable mobile hamburger menu
  - RGB color picker for mouse preview
  - Live countdown timer
  - Back to top button
  - Active nav highlight on scroll
